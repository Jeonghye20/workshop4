📝 설명
수업 전 단계

변수(Variable)는 프로그램에서 데이터를 저장하는 공간입니다. 파이썬에서는 = 기호를 사용해 변수를 선언하고 값을 할당합니다. 변수 이름은 영문자, 숫자, 밑줄(_)로 구성하며 숫자로 시작할 수 없습니다.

파이썬의 주요 자료형(Data Type)은 아래 코드 예시와 같습니다. type() 함수로 변수의 자료형을 확인할 수 있습니다.

name = "김민준" # str (문자열)
age = 15 # int (정수)
height = 165.5 # float (실수)
is_student = True # bool (논리)

print(type(name)) # <class 'str'>
print(type(age)) # <class 'int'>
