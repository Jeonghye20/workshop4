변수와 자료형의 핵심 개념을 학습해요
📌 변수(Variable)란?

변수는 데이터를 저장하는 공간입니다. 상자에 물건을 담듯이, 변수에 값을 넣어두면 나중에 꺼내 쓸 수 있어요.

파이썬에서는 = 기호로 변수에 값을 저장합니다. 변수 이름은 영문자·숫자·밑줄(_)로 구성하되, 숫자로 시작할 수 없어요.

name = "김민준" # 변수 name에 문자열 저장
age = 15 # 변수 age에 숫자 저장
print(name) # 출력: 김민준
print(age + 1) # 출력: 16
🗂 자료형(Data Type) 4가지

변수에 저장되는 값의 종류를 자료형이라고 합니다. type() 함수로 확인할 수 있어요.

int (정수)
age = 15
소수점 없는 숫자
float (실수)
height = 165.5
소수점 있는 숫자
str (문자열)
name = "김민준"
따옴표로 감싼 텍스트
bool (논리)
is_ok = True
True 또는 False
name = "김민준" # str
age = 15 # int
height = 165.5 # float
is_student = True # bool

print(type(name)) # <class 'str'>
print(type(age)) # <class 'int'>
print(type(height)) # <class 'float'>
